Proteksi RPC down: dana tetap terkunci di kontrak — RPC down cuma menunda tx, bukan menghilangkan dana. Watcher solver retry + multi-RPC fallback. Kalau timelock telanjur lewat (RPC mati total), kedua pihak tinggal refund() dan dana balik utuh. Guard: jangan reveal kalau sisa waktu < ambang aman.
Mekanik di balik demo ini diuji on-chain di Foundry — 10/10 test PASS (happy-path atomic, timeout-refund, permissionless claim, anti double-claim, fuzz 256×). Kontrak HTLC.sol + test ada di repo.
